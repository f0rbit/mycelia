# Getting Started with Mycelia

<Essay id="getting-started-guide" title="Getting Started Guide">
Setting up your first Mycelia-powered site is incredibly simple. This guide will walk you through the process.
</Essay>

## Quick Start

1. **Install the dependencies**:
   ```bash
   npm install @mycelia/ssr @mycelia/render @mycelia/parser
   ```

2. **Set up your Next.js app** with a catch-all route
3. **Create a `content/` folder** and add your MDX files
4. **Start writing** with Mycelia semantic tags

<Task id="setup-task" status="completed">
  Set up the basic Mycelia structure
</Task>

## Your First Content

Create an MDX file in your `content/` directory:

```mdx
---
title: "My First Post"
date: "2025-09-06"
---

# Hello Mycelia!

<Project id="my-project" status="active">
  This is my first project using Mycelia.
</Project>
```

The beauty of Mycelia is that every piece of content becomes discoverable and linkable. This tutorial connects to <Person id="demo-user">you</Person> and the broader <Collection id="main-content">knowledge base</Collection>.
