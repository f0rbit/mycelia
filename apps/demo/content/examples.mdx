## Example 4: Documentation

<Essay id="api-guide" title="API Integration Guide">
  Learn how to integrate with external APIs in your Mycelia project.

  ### REST APIs
  Most web services expose REST APIs for data access.

  ```typescript
  const response = await fetch('https://api.example.com/data')
  const data = await response.json()
  ```

  ### GraphQL APIs
  GraphQL provides a flexible query language for APIs.

  ```typescript
  const query = `
    query {
      user(id: "123") {
        name
        email
      }
    }
  `
  ```

  ### Best Practices
  - Cache responses when possible
  - Handle errors gracefully
  - Use TypeScript for type safety
  - Implement retry logic for failed requests

  For more patterns, see the <Link target="advanced-features">Advanced Features guide</Link>.
</Essay>
