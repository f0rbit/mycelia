# Quick Start

Get up and running with Mycelia in just a few minutes. This guide will walk you through creating your first digital garden.

## Installation

Install Mycelia using your preferred package manager:

```bash
# Using Bun (recommended)
bun add @mycelia/core @mycelia/parser @mycelia/render

# Using npm
npm install @mycelia/core @mycelia/parser @mycelia/render

# Using yarn
yarn add @mycelia/core @mycelia/parser @mycelia/render
```

> **Note**: `@mycelia/ssr` was merged into `@mycelia/render`. Use `@mycelia/render/nextjs` for SSR functionality.

## Create Your First Content

Create a new MDX file with Mycelia's custom tags:

```mdx filename="content/my-first-post.mdx"
---
title: My First Mycelia Post
date: 2024-01-01
---

# Building My Digital Garden

<Person name="Jane Doe" role="Developer">
  Passionate about creating interconnected knowledge systems.
</Person>

<Project title="Knowledge Base" status="in-progress">
  Building a personal knowledge management system using Mycelia.
  
  <Task name="Setup" status="completed">
    Initial project setup and configuration
  </Task>
  
  <Task name="Content Migration" status="in-progress">
    Migrating existing notes and documentation
  </Task>
</Project>

<Tag name="digital-garden" />
<Tag name="knowledge-management" />
```

## Setup Next.js Integration

Create a Next.js app with Mycelia:

```typescript filename="app/[...slug]/page.tsx"
import { MyceliaProvider } from '@mycelia/render/nextjs';
import { MarkdownParser } from '@mycelia/parser';

const provider = new MyceliaProvider({
  contentDir: './content',
  parser: new MarkdownParser()
});

export default async function Page({ 
  params 
}: { 
  params: { slug: string[] } 
}) {
  const content = await provider.getContent(
    params.slug.join('/')
  );
  
  if (!content) {
    return notFound();
  }
  
  return (
    <article>
      {content.render()}
    </article>
  );
}

export async function generateStaticParams() {
  const paths = await provider.getAllPaths();
  return paths.map(path => ({
    slug: path.split('/')
  }));
}
```

## Run Your Site

Start the development server:

```bash
# Using Bun
bun run dev

# Using npm
npm run dev
```

Visit `http://localhost:3000` to see your Mycelia-powered site!
